## Проверяем находимся ли мы на главной странице

```php
global $APPLICATION;

$isMainPage = $APPLICATION->GetCurPage(false) === SITE_DIR;

if ($isMainPage) {
    echo "Мы на главной странице.";
} else {
    echo "Мы на внутренней странице.";
}
?>
```

`$APPLICATION->GetCurPage(false):` Этот метод возвращает путь текущей страницы. Аргумент `false` означает, что Битрикс отрежет имя индексного файла (например, index.php), если он был явно указан в URL.

Если пользователь зашел на `site.ru/` → метод вернет `/`. Если пользователь зашел на `site.ru/index.php` → метод всё равно вернет `/`. `SITE_DIR`: Это константа, которая хранит путь к корню текущего сайта.
Обычно она равна просто `/`.
